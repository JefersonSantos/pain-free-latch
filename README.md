# Pain-Free Latch Blueprint

Sales page for the Pain-Free Latch Blueprint digital product.

## Local Development

```bash
npm install
npm run dev
```

Open:

```txt
http://localhost:3000
```

The sales page is also available at:

```txt
/sales-page
```

## Vercel

Set this environment variable in Vercel:

```txt
NEXT_PUBLIC_SITE_URL=https://your-domain.com
```

Checkout is configured in `app/sales-page/page.tsx`.
